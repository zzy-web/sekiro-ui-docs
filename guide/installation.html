# 环境支持

# 使用包管理器

我们建议您使用包管理器（如 NPM、Yarnopen in new window 或 pnpmopen in new window）安装 zView，然后您就可以使用打包工具，例如 Viteopen in new window 或 webpackopen in new window。

# 选择一个你喜欢的包管理器

# NPM
$ npm install element-plus --save

# Yarn
$ yarn add element-plus

# pnpm
$ pnpm install element-plus

如果您的网络环境不好，建议使用相关镜像服务 cnpmopen in new window 或 中国 NPM 镜像open in new window。

# 浏览器直接引入

直接通过浏览器的 HTML 标签导入 zView，然后就可以使用全局变量 zView 了。

根据不同的 CDN 提供商有不同的引入方式， 我们在这里以 unpkgopen in new window 和 jsDelivropen in new window 举例。 你也可以使用其它的 CDN 供应商。

# unpkg

<head>
  <!-- Import style -->
  <link rel="stylesheet" href="//unpkg.com/element-plus/dist/index.css" />
  <!-- Import Vue 3 -->
  <script src="//unpkg.com/vue@3"></script>
  <!-- Import component library -->
  <script src="//unpkg.com/element-plus"></script>
</head>

# jsDelivr

<head>
  <!-- Import style -->
  <link
    rel="stylesheet"
    href="//cdn.jsdelivr.net/npm/element-plus/dist/index.css"
  />
  <!-- Import Vue 3 -->
  <script src="//cdn.jsdelivr.net/npm/vue@3"></script>
  <!-- Import component library -->
  <script src="//cdn.jsdelivr.net/npm/element-plus"></script>
</head>

TIP

我们建议使用 CDN 引入 zView 的用户在链接地址上锁定版本，以免将来 zView 升级时受到非兼容性更新的影响。 锁定版本的方法请查看 unpkg.comopen in new window。
